By the end of this lesson, you will use Python variables and core types to hold input, output, and metadata.

## Prerequisites

Complete the Python setup lesson.

## Plain-English Explanation

Variables are named boxes. Each type tells your program what kind of value it can safely hold and how it behaves.

## Key Terms

### Data Type

Simple meaning:
The kind of value you are storing.

Real-world analogy:
Like choosing the right form field type (number, text, checkbox).

Technical meaning:
Python object categories such as `str`, `int`, `float`, `bool`, `list`, and `dict`.

Example:
Use `int` for lesson counts, `str` for text.

Why it matters:
Mixing types creates silent bugs before you ever touch AI components.
