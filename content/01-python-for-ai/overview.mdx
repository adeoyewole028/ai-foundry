# Module 01 Overview: Python for AI

This module builds the Python fluency needed for AI tooling, data pipelines, and API design.

## What you’ll learn

- Reliable Python setup, environments, and project hygiene.
- Data handling with core types, collections, files, and APIs.
- How to move from notebooks to a small API-based workflow.
- How to prepare reusable utilities for AI assistants and endpoints.

## Learning loop

You will move from explanation to exploration, then into implementation.
