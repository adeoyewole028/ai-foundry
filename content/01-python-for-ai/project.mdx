## Key Terms

### Contract

Simple meaning:
The agreed format between client and server.

Real-world analogy:
A service agreement: input format and output format.

Technical meaning:
JSON schema and endpoint semantics that remain stable over time.

Example:
`/analyze` expects `{ "text": "..." }` and returns `{ "ok": true, "meta": ... }`.

Why it matters:
Changes become deliberate, and clients avoid breaking unexpectedly.
