## Objective

By the end of this lesson, you will run experiments in notebooks and keep outputs reusable.

## Plain-English Explanation

Notebooks are great for trying ideas quickly, especially when testing prompts, parsing inputs, and comparing outputs.

## Key Terms

### Notebook Cell

Simple meaning:
One executable block.

Real-world analogy:
Like a page where you write one step at a time.

Technical meaning:
Ordered, interactive code or markdown block in a notebook runtime.

Example:
Cell order determines variable availability.

Why it matters:
Order-dependent execution can hide bugs when moving to scripts.
