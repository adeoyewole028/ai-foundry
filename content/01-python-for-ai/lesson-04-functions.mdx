# Functions

## Objective

By the end of this lesson, you will use reusable functions for text normalization and prompt preparation.

## Prerequisites

Practice with variables, types, lists, and dictionaries.

## Plain-English Explanation

Functions package behavior so your code becomes predictable and easy to test.

## Key Terms

### Function

Simple meaning:
Named block of code that can run with inputs.

Real-world analogy:
Like a recipe you run repeatedly with new ingredients.

Technical meaning:
A callable object that maps arguments to return values.

Example:
`def clean_text(text): ...`

Why it matters:
AI workflows become stable when preparation steps are centralized.

## Mental Model

```text
Input -> function -> transformation -> output
```

## Example

```python
def normalize(text: str) -> str:
    return " ".join(text.strip().split())
```

## Code Walkthrough

```python
def build_prompt(topic: str, max_chars: int = 120):
    cleaned = normalize(topic)
    return cleaned[:max_chars]
```

## Common Mistakes

- Functions doing too many things at once.
- Returning different shape responses from one function.
