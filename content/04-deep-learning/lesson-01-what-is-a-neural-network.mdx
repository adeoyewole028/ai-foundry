# What is a Neural Network?

## Objective

By the end of this lesson, you will describe a neural network as layered transformation.

## Prerequisites

Before this lesson, you should complete Module 03 and be comfortable with predictive workflows.

## Plain-English Explanation

A neural network is a chain of simple operations stacked together to learn complex patterns.

## Key Terms

### Neuron

Simple meaning:
A single decision unit.

Real-world analogy:
It is like one checkbox in a checklist that adds a little signal.

Technical meaning:
A weighted sum plus bias and optional activation producing one unit response.

Example:
One neuron deciding if text length suggests relevance.

Why it matters:
Many neurons together can represent richer decision boundaries.

## Mental Model

Imagine each layer as an editor that highlights different cues.

## Common Mistakes

- Treating every deep model as “black magic.”
- Ignoring what shape each layer expects.
- Assuming more layers always improve results.
