# Machine Learning

This module introduces predictive modeling and practical evaluation habits.

## How to use this module

1. Complete each lesson in order.
2. Finish the checklist items and answer reflection questions.
3. Complete the quiz to unlock the next curriculum module.
4. Finish both project components and record lessons learned.

## Focus areas

- Feature and label framing
- Split design and leakage prevention
- Regression and classification basics
- Tree models and ensembles
- Metrics and overfitting prevention
