## Key Terms

### Experiment Registry

Simple meaning:
Log of all adaptation trials.

Real-world analogy:
Research notebook for clinical protocol changes.

Technical meaning:
Track config, dataset version, metrics, and rollout status.

Example:
Storing each tuning run with pass/fail metadata.

Why it matters:
Auditability and reproducibility keep teams safe.

## Mental Model

Input data + config -> training -> validation -> rollout decision -> monitoring.

## Example

Fine-tune a concise support assistant and monitor for style drift.

## Code Walkthrough

Document:
- dataset schema
- training config
- eval criteria
- rollback strategy

## Common Mistakes

- No run IDs for reproducibility.
- Ignoring evaluation scope.
- Missing human oversight on post-change behavior.
