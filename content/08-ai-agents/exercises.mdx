# Module 08 Exercises

## Practical Drill

- Build one toy agent that handles a 3-step workflow with at least one tool.
- Add a memory payload that preserves goal, state, and last action.
- Add human approval for at least one potentially destructive step.

## Reflection Drill

- Write one paragraph describing one action your agent will never execute automatically.
- Map three metrics that best demonstrate whether it is safe and useful.
