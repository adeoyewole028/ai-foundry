# Data Visualization

## Objective

By the end of this lesson, you will be able to spot quality issues with simple charts.

## Prerequisites

Before this lesson, you should know numeric summaries and missing value basics.

## Plain-English Explanation

Charts show what tables hide. A distribution chart can reveal outliers quickly.

## Key Terms

### Outlier

Simple meaning:
An unusual value far from normal range.

Real-world analogy:
One bill of $1,000,000 in grocery receipts signals an error.

Technical meaning:
Points far from expected distributions, requiring treatment or investigation.

Example:
Histogram reveals impossible durations in an event dataset.

Why it matters:
Outliers can disproportionately shape model training.

## Mental Model

Plot before you transform. Visuals become your quality smoke test.

## Example

Use histograms and box plots to reveal skew and extreme values in target variables.

## Code Walkthrough

Create quick plots for numeric distributions and save a snapshot with notes.

## Common Mistakes

- Skipping visual checks when metrics look fine.
- Ignoring scale when chart scales hide outliers.
- Reading patterns without validating data generation assumptions.

## Exercise

Create two charts and explain one data issue each reveals.
