## Key Terms

### Baseline smoke test

Simple meaning:
A minimal check that prevents hidden setup failure.

Real-world analogy:
Turning on all circuits in a small room before expanding wiring.

Technical meaning:
Small representative run used to confirm architecture, shapes, logging, and reproducibility.

Example:
10-epoch run on a small subset with tracked metrics.

Why it matters:
It catches data and training breakages before expensive experimentation.

## Mental Model

Run small, verify, then scale.

## Example
