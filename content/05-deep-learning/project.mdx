## Key Terms

### Evaluation protocol

Simple meaning:
How you decide if the model is truly useful.

Real-world analogy:
Pre-flight checks before shipping an app version.

Technical meaning:
Split strategy, metrics, confusion diagnostics, and documented limitations.

Example:
Per-class precision and recall in a small visual dataset.

Why it matters:
Stakeholders need both performance and caveats.

## Mental Model
