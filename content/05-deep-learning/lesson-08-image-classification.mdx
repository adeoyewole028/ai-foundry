## Plain-English Explanation

Image classification maps image data to labels through learned visual patterns.

## Key Terms

### Channels

Simple meaning:
Color planes like red, green, blue.

Real-world analogy:
Three stacks of transparencies combined into one photo.

Technical meaning:
Tensor depth representing color/band dimensions.

Example:
An RGB image has 3 channels.

Why it matters:
Wrong channel assumptions break model behavior.

## Common Mistakes

- Ignoring class imbalance.
