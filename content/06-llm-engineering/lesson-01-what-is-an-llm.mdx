# What is an LLM?

## Objective

By the end of this lesson, you will explain LLM capabilities in practical product terms.

## Prerequisites

Before this lesson, you should understand deep-learning fundamentals.

## Plain-English Explanation

LLMs are large language models that predict likely text continuations and can power many product tasks.

## Key Terms

### Token prediction

Simple meaning:
The model guesses the next likely piece of text.

Real-world analogy:
Auto-complete in a messaging app.

Technical meaning:
Autoregressive decoding over a token sequence.

Example:
Suggesting a support response draft.

Why it matters:
Knowing the mechanism clarifies where errors can happen.

## Mental Model

LLM = strong language assistant with boundaries from prompts and context.

## Example

Drafting product copy and policy summaries from user input.

## Code Walkthrough

No code needed; compare model behavior across two short prompts.

## Common Mistakes

- Treating LLM as always truthful.
- Ignoring retrieval needs for facts.
- Over-promising from confidence in wording.
