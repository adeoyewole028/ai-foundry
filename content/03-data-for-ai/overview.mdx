# Module 03 Overview: Data for AI

This module teaches you how to prepare data so it becomes trustworthy input for AI systems.

## What you’ll learn

- How to inspect and profile raw data.
- How to clean, normalize, and validate data consistently.
- How to handle missing values without introducing bias.
- How to build simple visuals for data quality decisions.

## Learning loop

You will move from concept to practical checks, then into mini-project and portfolio work.
