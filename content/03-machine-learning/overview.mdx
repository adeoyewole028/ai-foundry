# Module 03 — Machine Learning

In this module, you will build your first practical predictive models.
You will learn to define inputs and targets, train, validate, and evaluate models,
and then ship a portfolio project that turns these ideas into a real workflow.

## Objectives

- Understand features and labels and why they matter.
- Train and evaluate simple machine learning workflows.
- Compare regression, classification, and tree-based strategies.
- Detect and reduce common failures like leakage and overfitting.

## What you will build

- Mini project: Job match score checker
- Portfolio project: Job match score predictor

## Progression expectations

This module is best taken after data preparation basics, with strong attention to data
quality and split discipline.
