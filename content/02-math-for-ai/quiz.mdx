## Example

Prompt: "Why is probability useful?"

Strong answer: "Probability helps AI express uncertainty, like which label is most likely, without pretending the prediction is guaranteed."

## Code Walkthrough

```python
scores = {"cat": 0.7, "dog": 0.2, "rabbit": 0.1}
print(max(scores, key=scores.get))
```

This picks the most likely class, but it does not make the prediction certain.
